## Vector

decleration syntax

vector<data_type>name_of_vector
eg: vector<int>a
  
*Modifiers*
```
push_back();
pop_back();
insert();
erase();
swap();
assign();
clear();
```
  
*Iterators*
```
begin();
end();
```
  
*capacity*
```
size();
max_size();
resize();
capacity();
empty();
shrink_to_fit();
```
